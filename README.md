# CODSOFT AI Internship Tasks

This repository contains all the tasks completed during my Artificial Intelligence internship at CodSoft.

---

## Task 1 — Chatbot with Rule-Based Responses
A general purpose AI chatbot built using pure Python.
- Pattern matching and keyword detection
- Randomized responses for natural conversation
- Tells current time and date
- Responds to user mood
- Tells jokes
- No external libraries used

---

## Task 2 — Tic-Tac-Toe AI
An unbeatable Tic-Tac-Toe AI agent built using Python.
- Minimax algorithm for optimal move calculation
- Human vs AI gameplay
- Validates all user inputs
- Detects win, loss and draw conditions
- No external libraries used

---

## Task 3 — Image Captioning
An Image Captioning AI that combines Computer Vision and Natural Language Processing.
- ResNet50 for image feature extraction
- Microsoft GIT transformer model for caption generation
- Supports multiple image uploads in one run
- Generates accurate natural language captions
- Built using PyTorch and HuggingFace Transformers

---

## Task 4 — Recommendation System
A content-based Movie Recommendation System built using Machine Learning.
- TF-IDF Vectorization for analyzing movie tags
- Cosine Similarity for finding similar movies
- Dataset of 30 movies across multiple genres
- Returns top 5 similar movies instantly
- Built using Scikit-learn and Pandas

---

## Task 5 — Face Detection and Recognition
An AI application that detects and recognizes faces in images.
- Haar Cascade classifier for face detection
- Detects multiple faces in a single image
- Draws bounding boxes around detected faces
- Supports user uploaded images
- Built using OpenCV and Python
